Curriculum

Week 1 ‚Äî The Hacker's Toolkit

Core Topic: Setting up your virtual lab (Kali/Parrot) and mastering essential Linux command-line navigation (ls, cd, cat, grep, find).
Key Concept: Understanding the "Five Phases of Hacking" as a mental model for every engagement.

Hands-On Goal: Successfully install a VM and use basic commands to find a hidden flag in the filesystem.

Deliverable: Submit the flag found on the practice machine.

Week 2 ‚Äî Mapping the Terrain with Nmap

Core Topic: Active network scanning. Understanding what ports are and what they tell you.
Key Tool: Nmap.

Hands-On Goal: Scan a target IP address to identify open ports, running services, and the operating system version.

Deliverable: A screenshot of your Nmap scan results showing the services on a target.

Week 3 ‚Äî Intercepting Web Traffic

Core Topic: Introduction to web application hacking and how HTTP requests work.
Key Tool: Burp Suite (Community Edition).

Hands-On Goal: Configure your browser to use Burp Suite as a proxy, intercept a login request, and modify it before sending it to the server.

Deliverable: A screenshot showing a modified HTTP request in Burp Suite's Repeater tab.

Week 4 ‚Äî SQL Injection (SQLi)

Core Topic: Exploiting databases through web applications. Focus on understanding how user input can be turned into a database query.
Key Tool: SQLMap.

Hands-On Goal: Use SQLMap to automate the process of enumerating the databases on a vulnerable web application.

Deliverable: The name of a hidden database found using SQLMap.

Week 5 ‚Äî Cross-Site Scripting (XSS)

Core Topic: Exploiting trust by injecting malicious scripts into web pages viewed by other users. Focus on simple Reflected XSS.
Key Tool: Browser Developer Tools.

Hands-On Goal: Craft a basic JavaScript payload to create an alert pop-up on a vulnerable webpage.

Deliverable: The working XSS payload you created.

Week 6 ‚Äî Gaining a Foothold with Metasploit

Core Topic: Using an exploitation framework to gain initial access to a target system.
Key Tool: Metasploit Framework.

Hands-On Goal: Find and configure a pre-packaged exploit to gain a remote shell on a known vulnerable machine (e.g., Metasploitable2).

Deliverable: A screenshot of your active Meterpreter session on the target machine.

Week 7 ‚Äî Cracking the Code

Core Topic: Understanding password hashing and basic cracking techniques.
Key Tool: John the Ripper (JtR) or Hashcat.

Hands-On Goal: Use a common wordlist to crack a list of provided password hashes (e.g., MD5, SHA-256).

Deliverable: The list of cracked passwords.
